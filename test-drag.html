ParaNet 拖拽测试

测试页面: http://localhost:3000/develop

测试步骤:

  1. 打开 http://localhost:3000/develop
  2. 等待页面加载完成
  3. 选择任意项目(如果有的话)
  4. 切换到"拓扑工作区"标签页
  5. 从左侧"设备图例"拖拽任意设备到右侧画布

预期行为:

1. 拖拽开始: 图标能够被拖动(draggable=true 已设置)
2. 画布响应: 右侧画布应显示蓝色虚线高亮边框
3. 松手创建: 释放鼠标时应自动创建设备节点

需要在浏览器控制台手动检查:

// 1. 检查拖拽源元素
document.querySelectorAll('[draggable="true"]')

// 2. 检查事件监听器
const canvas = document.querySelector('.topologyCanvas')
console.log('Canvas element:', canvas)

// 3. 监听拖拽事件
window.addEventListener('dragstart', e => console.log('🎯 DRAG START', e.dataTransfer.types))
window.addEventListener('dragover', e => console.log('📍 DRAG OVER', e.target))
window.addEventListener('drop', e => console.log('✅ DROP', e.dataTransfer.getData('text/plain')))

// 4. 检查全局状态
console.log('Active dragged device:', window._debugActiveDraggedDevice)

常见问题诊断:

问题1: 拖拽无法开始
→ 检查 draggable 属性和 onDragStart 事件
问题2: 画布不高亮
→ 检查 onDragOver 事件和 preventDefault()
问题3: 松手不创建节点
→ 检查 onDrop 事件和 dataTransfer 数据
